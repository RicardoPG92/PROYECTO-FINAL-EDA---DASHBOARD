  ANÁLISIS EXPLORATORIO Y PRESENTACIÓN MEDIANTE PANEL DE DATOS DE UNA SERIE DE DATOS DE VENTAS DE SUPERMERCADOS
   
   DESCRIPCIÓN DE LO TRABAJOS PREVIOS REALIZADOS:
      ESTE DATASET CONTIENE UN ARCHIVO CSV QUE DEBEMOS IMPORTAR Y TRATAR EN UN FICHERO EXCEL PARA DARLE UN FORMATO INTELIGIBLE Y EN QUE PODAMOS TRABAJAR. 
      LA IMPORTACIÓN A EXCEL DE LOS DATOS LA HACEMOS MEDIANTE LA FUNCIONALIDAD DE OBTENER DATOS DESDE CSV EN LA PESTAÑA DATOS. cARGAMOS LOS DATOS DEJANDO QUE EL SISTEMA DETECTE EL TIPO DE DATOS EN FUNCIÓN DE LOS 200 PRIMEROS REGISTROS. 
      LA BBDD RESULTANTE CONTIENE 15 COLUMNAS Y 5000 FILAS CON DIFERENTES REGISTROS. 
      ESTE DATASET CONTIENE REGISTROS DE TRANSACCIONES DE VENTAS DE UNA EMPRESA DEDICADA APROVISIONAMIENTO DE PRODUCTOS EN CATEGROÍAS COMO LA ELECTRÓNICA, MOBILIARIO O ACCESORIOS DEL HOGAR. LA EMPRESA OPERA EN VARIAS REGIONES: NORTE AMÉRICA, SUDAMÉRICA, EUROPA Y ASIA. CADA REGISTRO CONTIENE DIFERENTES DETALLES DEL PROCESO DE VENTA INCLUYENDO NÚMERO DE TRANSACCIÓN, SEGMENTO DE CLIENTES, REPRESENTANTE DE VENTAS, EQUIPO, CATEGORÍA DE PRODUCTO, ETC…
